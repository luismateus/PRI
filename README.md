Commands to run in terminal:

Ad hoc search:

$ python createIndex.py 

this script will take a while (the first time) as it is responsible for creating the index and the named entities dictionary

$ python searchCollection.py <keywords>

we assume that the keywords have no errors



Classification:

$ python classification.py <text>

we assume that the given natural language has no errors



Entity Analysis:

$ python statisticalAnalysis.py
